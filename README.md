\# House Price Prediction



\## Overview

This project predicts house prices using machine learning techniques based on various features such as living area, number of bedrooms, and house condition.



\## Features

\- Data preprocessing and cleaning

\- Feature engineering (Total area, House age)

\- Outlier removal

\- Linear Regression model



\## Tech Stack

\- Python

\- Pandas

\- Scikit-learn

\- Matplotlib / Seaborn



\## Results

\- Improved model performance by \~13%

\- Mean Absolute Error (MAE): \~107,000
